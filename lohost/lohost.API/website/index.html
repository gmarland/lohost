What is lohost?

lohost is a website client that turns your computer into a server so you can publish your website without using a hosting company. Once you’ve set up lohost on your computer and reserved a URL, your website will be live whenever you're running the lohost software.

How does lohost protect your privacy?

Unlike most hosting companies, lohost gives you complete control over your website data and you can reserve a URL without needing to share any personal information.

How does lohost give you more control?

lohost lets you control when your website is live, as you can remove it from the internet at any time by simply turning the software off. You can also install lohost on more than one computer and use any of them to power your website.

How do you get started?

You can get started by following these steps:

  • Download the website client installer on your Windows computer
  • Check that your setup.exe is unblocked
  • Select a location
  • Use a command prompt to run the .exe
